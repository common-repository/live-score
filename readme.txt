== Installation ==

1. Upload the folder `/live-score/` to your WP plugin folder `/wp-content/plugins/` directory
2. The path must look like this: `/wp-content/plugins/lives-core/`
3. Create a new page with a name `Live Score` from Pages section. (Page name must be  `Live Score`)
4. Activate the plugin through the 'Plugins' menu in WordPress admin
5. You are ready, check your live score page.

Thanks for installing!
